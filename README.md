# LibrAI
IA para Tradução de Libras em Tempo Real

Status: Protótipo Funcional

Objetivo

Desenvolver uma solução baseada em visão computacional capaz de reconhecer sinais básicos da Língua Brasileira de Sinais 
(Libras), convertendo-os automaticamente em texto e áudio para facilitar a comunicação.

Principais recursos

✅ Reconhecimento de sinais pela webcam

✅ Processamento em tempo real

✅ Conversão dos sinais para texto

✅ Resposta por voz (Text-to-Speech)

✅ Interface simples para demonstração

Tecnologias
Python
OpenCV
MediaPipe
NumPy
Text-to-Speech
Visão Computacional
Competências demonstradas
Inteligência Artificial
Processamento de imagens
Reconhecimento de padrões
Manipulação de vídeo em tempo real
